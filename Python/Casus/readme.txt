Te program uses a MSQL database, make sure you have installed and are running the database (the export of the database can be found in the folder named SQL).

Make sure you have installed the following Python packages before running the code:
-   pyttsx3
-   gtts
-   playsound
-   sys
-   random
-   time
-   datetime
-   mysql.connector

Run the client.py to run the program.

screens.py:     Is used to display most of the screens within the program.
screenCompo.py: Is used to compose certain aspects of the pages.
req.py:         Is used to get/send data from/to the database.
